Software and Tools

MAST / MAST ML

MAST provides tools and workflows that make machine learning easier for materials scientists.

Materials Commons
Materials Commons is a platform for organizing, collaborating, publishing and sharing research data.
PRISMS-PF
PRISMS-PF is a powerful, massively parallel finite element code for conducting phase field and other related simulations of microstructural evolution.
MDF
The MDF is a set of data services built specifically to support materials science researchers.
Forge
Forge is the Materials Data Facility Python package to interface and leverage the MDF Data Discovery service. Forge allows users to perform simple queries and facilitiates moving and synthesizing results.
MDF Connect
The MDF Connect is the Python library for easily querying, accessing, and using the data in the Materials Data Facility.
4CeeD
The 4CeeD Uploader and Curator are web tools used to upload, organize, and manage scientific data files and metadata.
